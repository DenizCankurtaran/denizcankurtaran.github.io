## Primitive Datentypen
| Typ       | Speicher |
| char      | 1 Byte   |
| short     | 2 Bytes  |
| int       | 4 Bytes  |
| long      | 4 Bytes  |
| long long | 8 Bytes  |
| float     | 4 Bytes  |
| double    | 8 Bytes  |
| bool      | 1 Byte   |

Keyword *unsigned* vor jedem Typen entfernt das sign bit. *sizeof* function liefert die byte Größe.
```
sizeof(int); //4
```
